    Day1

    杂题选讲

    Subset

    给出 \(n\) 个数，要求求出两个不相交的subset，使得他们的和一样。

    \(n, a_i \leq 10^5\)

    \(2^{22} = 4194304 \geq 22 \times 10^5\)。

    根据鸽笼原理，前 22 个数中一定存在两个集合的和相同，找出这两个集合然后去掉公共部分即可。

    City Brain

    给定 \(n\) 点 \(m\) 条边的无向图，边的长度都为 \(1\)，限速都为 \(1\)，你有 \(k\) 元，每次可以花费 \(1\) 元将任意一条边的限速提高 \(1\)。从 \(s_1\) 到 \(t_1\) 和从 \(s_2\) 到 \(t_2\)，求用时之和的最小值。

    \(1 \leq n \leq 5000, 0 \leq m \leq 5000, 0 \leq k \leq 10^9\)。

    两个路径的公共部分一定是连续的一段，枚举这段的起点和终点，对于非公共部分，边数最少的更优。

    对于公共部分长度相同的方案，只需要考虑非公共部分边数最少的方案，然后贪心地分配钱。

    花钱的收益是递减的，因此可以二分最后花钱的收益不小于一个值，计算答案。

    复杂度 \(O(n^2 + n\log n)\)。

    Mr. Panda and Typewriter

    你有一个打字机，可以进行三种操作：

    1. 花费 \(X\) 代价在末尾添加一个字符。
    2. 花费 \(Y\) 代价复制一段已经打印的字符。
    3. 花费 \(Z\) 代价粘贴。

    求打印出字符串 \(S\) 的最小代价。

    \(1 \leq n \leq 5000, 1 \leq X, Y, Z \leq 10^9\)。

    设 \(f_{i, j}\) 表示打了前 \(i\) 个字符，剪切板是最后 \(j\) 个字符的最小代价。

    1. \(f_{i, j} \to f_{i + 1, 0}\) 打一个字符。
    2. \(f_{i, j} \to f_{i, k}\) 复制。
    3. \(f_{i, j} \to f_{k, j}\) 粘贴，中间用操作 1 填满。

    Bookcase Solidity United

    \(n\) 层的书架，从上到下第 \(i\) 层最多能承受小于 \(a_i\) 个球，初始是空的。如果一层的球 \(x \geq a_i\)，这层书架会被击穿，然后球会下落，\(\lfloor \dfrac{x}{2} \rfloor\) 的球会落到最近的没被击穿的书架，其余的会消失。分别求击穿前 \(k\) 层的最小代价。

    \(1 \leq n \leq 70, 1 \leq a_i \leq 150\)。

    设 \(f(l, r, k)\) 表示击穿 \([l, r]\)，剩 \(k\) 个球的最小代价。

    1. \(f(l, r + 1, \max\{k, a_{r+1}\} / 2) = f(l, r, k) + \max\{0, a_{r+1} - k\}\)
    2. \(f(l, p, k_1 + k_2) = f(l, r, k_1) + f(r+1, p, k_2)\)

    Infimum of Paths

    \(n\) 点 \(m\) 条边的无向图，每条边有边权 \(w(0 \leq w \leq 9)\)，一条路径 \((w_1, w_2, \dots, w_l)\) 的权值是 \(0.w_1w_2\dots w_l\)，求 \(0\) 到 \(1\) 的最短路。答案对 \(10^9+7\) 取模。

    \(2 \leq n \leq 2000, 1 \leq m \leq 4000\)。

    分别考虑有环的和无环的路径。

    维护从 \(0\) 开始走 \(k\) 步能到达的点集，只保留最短路径，并记录前驱。

    走 \(n + 1\) 步，中间到了 \(1\) 就更新答案，这是不包含环的。

    对于包含环的路径，\(n + 1\) 步一定遍历了整个环，所以直接根据前驱回溯即可。

    Bytelandia States Union

    网格图，从 \((x, y)\) 走向 \((x+1, y), (x-1, y), (x, y+1), (x, y-1)\) 的代价分别是 \(2xy^2+2y^2+x^2, -2xy^2+2y^2+x^2, 2x^2y+2x^2+y^2, -2x^2y+2x^2+y^2\)。\(n\) 次询问 \((x_1, y_1)\) 走到 \((x_2, y_2)\) 的最小代价。 \(1 \leq n \leq 50000, 1 \leq x_1, y_1, x_2, y_2 \leq 10^9\)。

    Mr. Panda and Cactus

    给定 \(n\) 点 \(m\) 条边的仙人掌，需要给点染上 \(k\) 种颜色，所有颜色必须用至少一次。每条边 \((u_i, v_i)\) 有边权 \(w_i\)，一种染色方案的代价为 \(\sum w_i[u_i = v_i]\)，求最小代价。

    \(1 \leq n \leq 10^5, 0 \leq m \leq 2\times 10^5, 1 \leq k \leq \min\{n, 1000\}\)。

    最优方案的每种颜色一定会形成恰好一个联通块，转化为删去一些边形成 \(k\) 个联通块的代价。

    • 对于树边，删掉一条会增加一个联通块；
    • 对于一个环，删掉第 \(i\) 条边会增加 \(i-1\) 个联通块。

    对于每条树边和每个环动态规划即可。

    Query On A Tree 17

    \(n\) 个点的树，1 为根，点权 \(a_i\) 初始都为零。\(q\) 次操作，有如下两种： 1. 1 u：子树点权加 1。 2. 2 u v：链点权加 1。

    每次操作后询问深度最小的带权重心。

    \(n, q \leq 10^5\)。

    二次剩余

    勒让德符号

    \[ \left({\frac {a}{p}}\right) = \begin{cases} 0, &a \equiv 0 \pmod p\\ +1, &\text{a 是二次剩余}\\ -1, &\text{a 不是二次剩余} \end{cases} \] \[ \left({\frac {a}{p}}\right) \equiv a^{\frac{p-1}{2}} \pmod p \] #### Cipolla 求解 \(x^2 \equiv n \pmod p\)，其中 \(p\) 是奇素数。

    1. 随机找数 \(a\) 满足 \(\left({\frac {a^2-n}{p}}\right)=-1\)。
    2. \(x \equiv (a + \sqrt{a^2-n})^{\frac{p+1}{2}} \pmod p\)

    任意模数

    求解 \(x^2 \equiv n \pmod {p^e}\)，其中 \(p\) 是奇素数。 1. \(x_0^2 \equiv n \pmod p\) 2. \((x_0^2-n)^e \equiv 0 \pmod{p^e}\) 3. \((x_0-\sqrt n)^e(x_0+\sqrt n)^e \equiv 0 \pmod{p^e}, (x_0\pm\sqrt n)^e = u \pm v\sqrt n\) 4. \(u^2 - v^2n \equiv 0 \pmod{p^e}\) 5. \(\sqrt n \equiv uv^{-1} \pmod{p^e}\)

    求解 \(x^2 \equiv n \pmod {2^e}\)。 1. \(x_e^2 \equiv a \pmod{2^e}\) 2. \(x_{e+1} \equiv x_e + 2^{e-1}\left[\dfrac{x_e^2-a}{2^e} \text{ is prime}\right] \pmod{2^{e+1}}\)

    中国剩余定理合并。
